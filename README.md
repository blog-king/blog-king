## api文档
[使用apidoc](https://apidocjs.com/)

* 安装apidoc  `npm install apidoc -g`
* 根目录执行 `apidoc -i app/Http/Controllers/ -o public/apidoc`, （可自行修改输出目录）
* 查看 https://{yourHost}/apidoc
